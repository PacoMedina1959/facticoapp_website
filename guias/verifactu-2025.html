VeriFactu 2025: Guía Completa para Autónomos

Actualizado: Octubre 2025 | Tiempo de lectura: 8 minutos

¿Qué es VeriFactu?

VeriFactu es el nuevo sistema de facturación electrónica obligatorio en España, establecido por el Real Decreto 1007/2023. Es un sistema que permite a la AEAT verificar la autenticidad e integridad de las facturas emitidas por empresas y autónomos.

Objetivo principal: Combatir el fraude fiscal mediante la verificación en tiempo real de las facturas emitidas y recibidas.

El sistema funciona mediante registros encadenados criptográficamente, donde cada factura se vincula con la anterior mediante un hash SHA-256, creando una cadena inmutable de registros.

¿Quién debe usar VeriFactu?

Obligatorio para:

  • Empresas y autónomos que facturen más de 6.000€ anuales
  • Profesionales en estimación directa (régimen general)
  • Empresarios en estimación objetiva (módulos) con facturación superior a 6.000€
  • Actividades económicas que requieran facturación

Exentos:

  • Autónomos con facturación inferior a 6.000€ anuales
  • Actividades exentas de IVA (salvo excepciones)
  • Operaciones no sujetas a facturación
Importante: Aunque estés exento, es recomendable implementar VeriFactu si planeas crecer o si trabajas con empresas que sí están obligadas.

¿Cuándo entra en vigor?

Calendario de implantación actualizado:

  • 29 de julio de 2025: Fin del plazo para desarrolladores de software
  • 1 de enero de 2026: Empresas (todas las empresas)
  • 1 de julio de 2026: Autónomos y comunidades de bienes
Buenas noticias: Los autónomos tienen hasta julio de 2026 para implementar VeriFactu.

¿Cómo funciona VeriFactu?

1. Registro de facturas

Cada factura emitida debe registrarse en el sistema VeriFactu con:

  • Datos del emisor y receptor
  • Importe total y base imponible
  • Fecha de emisión
  • Número de factura
  • Hash criptográfico

2. Cadena de registros

Los registros se encadenan criptográficamente:

  1. Se genera un hash SHA-256 de la factura
  2. Se incluye el hash de la factura anterior
  3. Se crea un registro único e inmutable
  4. Se envía a la AEAT (opcional pero recomendado)

Implementación práctica

Opciones disponibles:

  1. Software de facturación compatible (recomendado)
  2. API de la AEAT (para desarrolladores)
  3. Portal web de la AEAT (manual, no recomendado)

Requisitos técnicos:

  • Certificado digital válido
  • Conexión a internet
  • Software compatible con VeriFactu
  • Registro en el sistema de la AEAT
Con Fáctico: VeriFactu está completamente integrado. Solo necesitas configurar tu certificado digital y el sistema se encarga de todo automáticamente.

Sanciones y multas

Sanciones por incumplimiento de VeriFactu:

  • 50.000€ fijos: Por cada ejercicio en que se use software no adaptado a VeriFactu
  • 150.000€ fijos: Para fabricantes de software no certificado (por ejercicio y tipo de software)
  • 1.000€ fijos: Por cada sistema comercializado sin certificado

Requisitos técnicos obligatorios:

  • Registro secuencial de cada operación
  • Generación de hash criptográfico único
  • Encadenamiento de registros mediante hash
  • Código QR estandarizado en facturas
  • Capacidad de remisión telemática a la AEAT
Importante: Las sanciones son muy elevadas. Es fundamental usar software certificado y adaptado a VeriFactu.

Ventajas de VeriFactu

Para el autónomo:

  • Mayor seguridad en las facturas
  • Reducción del fraude
  • Mejor control fiscal
  • Automatización de procesos

Para la AEAT:

  • Control en tiempo real
  • Detección de fraudes
  • Mejor recaudación
  • Reducción de inspecciones

Obligación: No está obligado, pero es recomendable.
