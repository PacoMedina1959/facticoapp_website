2. OBJETO Y FINALIDAD DEL SITIO WEB

El sitio web y la aplicación Fáctico tienen como objeto:

  • Facilitar el acceso y uso de una aplicación web de facturación electrónica y contabilidad básica
  • Permitir a autónomos y pequeñas empresas:
    • Gestionar facturas emitidas y recibidas
    • Llevar control de cobros y pagos (Caja)
    • Generar asientos contables automáticos
    • Cumplir obligaciones fiscales (modelos 303, 347, Verifactu)
    • Generar facturas electrónicas (PDF, Facturae, UBL)
    • Gestionar remesas SEPA y conciliación bancaria
    • Trabajar en equipo con control de roles y permisos
